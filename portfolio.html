bioMérieux DevOps infrastructure

GitLab CI/CD Jenkins Docker VMware vSphere Prometheus Grafana

DevOps infrastructure for 3P Connect, a web application for environmental control and traceability in pharmaceutical and industrial settings.

  • Migrated about 20 Jenkins pipelines to GitLab CI/CD, verified that artifacts matched Jenkins build outputs, and retired three Jenkins VMs in favor of shared Linux runners.
  • Dockerized Robot Framework test execution and moved three test suites from a dedicated Windows runner to shared Linux runners.
  • Built three GitLab CI/CD pipelines: one to update Windows VM templates, one to deploy 3P Connect components to a VM, and one to create a ready-to-use integration VM in under one hour.
  • Deployed Prometheus/Grafana dashboards for four runners, 12 integration VMs, CI/CD metrics, and load tests.

Shows: CI/CD migration, Dockerized test infrastructure, VM provisioning automation, and observability.

ft_irc

C++ sockets non-blocking I/O protocols

C++ IRC server handling multiple clients over TCP with non-blocking sockets.

  • I implemented the networking layer and core IRC commands and replies: PASS, PING, NICK, QUIT, USER, and the welcome reply.
  • The server uses non-blocking sockets with a single poll-based event loop.
  • It supports IRC commands and channel modes such as KICK, INVITE, TOPIC, and MODE.
  • Shared server state tracks users, channels, permissions, and message routing.

Shows: network I/O, protocol implementation, and stateful server design.

minishell

C Unix parser processes teamwork

Unix shell in C implementing a subset of Bash parsing, expansion, and process execution.

  • I implemented the lexer and parser for quotes, environment expansion, $?, pipes, redirections, heredocs, and append mode.
  • The shell resolves commands through PATH and uses fork, execve, pipes, dup2, and wait handling for execution.
  • It supports the required builtins: echo, cd, pwd, export, unset, env, and exit.
  • It reproduces expected interactive behavior for Ctrl-C, Ctrl-D, and Ctrl-\.

Shows: Unix process control, parsing, file descriptors, pipes, and signals.

Fixme

Java Maven non-blocking sockets concurrency FIX-inspired protocol

Multi-module Java market-messaging simulation using a custom, FIX-inspired protocol over non-blocking TCP sockets.

  • Built Broker, Router, and Market components as independent runnable Maven modules.
  • Used non-blocking sockets and executor-based message handling.
  • Implemented FIX-like message validation, checksums, routing IDs, market registration, and order responses.
  • Designed router behavior around routing tables without embedding business logic.

Shows: non-blocking sockets, concurrent architecture, protocol parsing, and distributed component design.
